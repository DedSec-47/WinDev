What is the Windows API?

The Windows API (WinAPI) is Microsoftâ€™s core set of application programming interfaces for interacting with the Windows operating system.

Common API Categories

  • Kernel32: Process & memory management
  • User32: GUI elements (windows, menus, dialogs)
  • GDI32: Graphics drawing interface
  • Advapi32: Registry, services, and security
  • Ws2_32: Networking (Winsock)

Example: MessageBox


#include <windows.h>

int main() {
    MessageBox(NULL, "Hello from WinAPI!", "MyApp", MB_OK);
    return 0;
}
